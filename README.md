# right-leyu-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容各自独立，不针对任何特定域名或网站。

## 项目简介

这是一个以 HTML 页面为主的归档仓库。页面均以静态形式存放，可直接通过浏览器打开查看。仓库本身不提供任何网页服务或动态功能，仅作为页面文件的存储与版本管理。

## 目录说明

- 根目录下的 HTML 文件为独立页面，文件名即页面标识。
- 每个页面均为自包含的 HTML 文件，不依赖外部资源（除非在页面内显式引用）。
- 目录结构保持扁平化，便于快速定位和引用。

## 页面归档说明

- 所有归档页面均为非交互式或有限交互式的静态文档。
- 页面内容可能涉及信息展示、工具界面或其他静态功能，不包含后端逻辑。
- 归档页面可能不定期更新或替换，请以仓库中的最新版本为准。
- 若某个页面不再维护，将在文件名中添加 `_archived` 后缀或移入 `archive/` 子目录。

## 维护说明

- 本仓库仅由维护者负责更新，不接受外部直接提交的页面文件。
- 如发现页面内容错误或链接失效，欢迎通过 Issues 提出，但请注意 Issues 仅用于技术问题反馈，不用于内容修改请求。
- 仓库可能不定期进行整理和清理，包括删除过时或无关的页面。
- 本仓库不保证任何页面的长期可用性，也不承担页面内容引起的任何责任。

## 许可

本仓库中的 HTML 页面文件遵循 [MIT 许可证](LICENSE)。请在使用或引用时遵守相关条款。
